Using the Chemical Terms language in Pipeline Pilot

 

Evaluating Chemical Terms functions can be done via the Chemical Terms Pipeline Pilot node. This type of node accepts a molecule as an input, and performs the specified calculation on it. The calculation is defined as a Chemical Terms function. The function can be specified in the Expression box of the Parameters section of the node.The result of the calculation is usually passed to an output node.

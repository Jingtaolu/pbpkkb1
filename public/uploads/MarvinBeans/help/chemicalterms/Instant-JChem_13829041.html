Chemical Terms : Instant JChem

Instant JChem (IJC) is a desktop application that allows scientists to create, manage and analyse chemical structures and their corresponding chemical and biological data.

In IJC, chemical properties can be calculated through Chemical Terms expressions that can be either added to an existing database as a new field or used as an additional query expression.

Introduction

 

The Refractivity Plugin is able to calculate molar refractivity. This descriptor is strongly related to the volume of the molecules and to London dispersive forces that play crucial part in drug-receptor interactions. Our calculation is based on the atomic method proposed by Viswanadhan et al. 

The calculation results with the value of refractivity (the dimension is 106⋅[m3⋅mol-1]) can be displayed in 2D or 3D.

Options

 

Different calculation parameters can be set in the Refractivity Options window:

  • Decimal places: setting the number of decimal places with which the result value is given.
  • Type: sets the type of the calculation
    • Increments: displays the increments given by atoms.
    • Refractivity: calculates the value of the molar refractivity
  • Increments of hydrogens: displays the increments given by hydrogens.
  • Display in MarvinSpace: if checked, the result window opens as a 3D MarvinSpace viewer; otherwose it is shown in 2D.

 

The numbers in brackets refer to the refractivity sum of the implicit hydrogen atoms.

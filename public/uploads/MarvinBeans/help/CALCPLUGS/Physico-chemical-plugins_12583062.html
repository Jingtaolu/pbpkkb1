Calculator Plugins : Physico-chemical plugins

The group of physico-chemical plugins contains tools for predicting different physical/chemical properties for molecules. They are grouped into bundles based on the property they calculate:

  • HLB Predictor
  • Isoelectric Point Plugin
  • logD Plugin
  • logP Plugin
  • Major Microspecies Plugin
  • NMR Predictor
  • pKa Plugin
  • Solubility Predictor
  • Stereo Analysis - calculating stereo descriptors
  • Stereoisomer Generator Plugin
  • Tautomer Generation Plugin

Calculator Plugins : Structural property plugins

The group of structural property plugins contains tools for calculating different 2D or 3D structural descriptors. They are grouped into bundles based on the type of descriptors they calculate:

  • Elemental Analysis Plugin
  • Geometrical Descriptors Plugin
  • Hückel Analysis Plugin
  • Hydrogen Bond Donor/Acceptor Plugin
  • Molecular Surface Area Plugin (3D)
  • Polar Surface Area Plugin (2D)
  • Refractivity Plugin
  • Resonance Plugin
  • Structural Frameworks Plugin
  • Topological Analysis Plugin

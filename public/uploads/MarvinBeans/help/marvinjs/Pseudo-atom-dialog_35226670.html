Marvin JS : Pseudo atom dialog

In pseudo atom dialog you can create a pseudo atom by setting its title.

 

How to draw a pseudo atom

  • Click the Pseudo atom button in the Query atom combo-box on the Atom Toolbar.
  • Fill the text field in the Pseudo atom dialog window.
  • Click OK.
  • The Pseudo atom with the inserted text will appear on the tip of the mouse cursor.

Marvin JS : Atom mapping

Unlike atom indices, map numbers remain constant during editing the molecule. 

Atom map visibility should be turned on in View settings dialog to display the atom map numbers.
  • Setting map numbers to individual atoms. 
  • Assigning the same number to corresponding atoms.
  • Auto mapping (only in reaction).

How to set map numbers to individual atoms

 

 

 

You can add map number to every single atom manually via the Atom properties dialog.  
Open the Basic Tab in the dialog, activate Map field by clicking the Padlock icon next to the imput field. Type the number then click OK. 

You can change the number in this menu anytime.

 

 

 

 

 

 

 

How to delete map numbers from individual atoms

 


To remove atom map numbers from atoms you can delete the Map value in Atom propertied dialog by closing the Padlock next to the imput field.

 

How to assign the same number to corresponding atoms

 

Select the smart Reaction tool from the Tools toolbar, then push the left mouse button on the atom you would like to map, and drag to the other atom which should get the same map number.  (Golden brown feedback helps the process.) After releasing the mouse button, the atom map number appears on the lower-right corner of the selected atoms.

 

When there is no map number on any of the atoms, both of the atoms get the smallest number which isn't represented on the canvas.

 

In some cases you need to map atoms with the same number as another atom has.
Select the smart Reaction tool from the Tools toolbar, then push the left mouse button on the atom you would like to map, and drag to the other atom which sould get the same map number. When the second atom gets a golden-brown feedback release the mouse button.

 

 

 


The same happens when you drag from the atom which has no map number to another which has one. Both of the atoms will get the same map number as the latter one had.

 

 

Overwriting the atom map number

 


Dragging from an atom which has already an atom map number to another atom with an atom map, the latter atom will get the same number as the former atom has.

 

Deleting the atom map number

 

Dragging from an atom with an atom map number to another atom with the same number results that both of the map numbers will be deleted from the canvas.

 

Auto-mapping

There is another option to assign map numbers to the corresponding atoms in a reaction.

Click the Auto Map button. In this case every atom in the reaction would get atom map number automatically. This feature is only available with Standardizer license.

Please note, that Auto-map function only works for structures which are in the same reaction. (In other words: there should be only one reaction (one arrow) on the canvas where every molecule would be a reactant or a product of it.)

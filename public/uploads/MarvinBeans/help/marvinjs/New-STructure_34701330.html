Marvin JS : New STructure

Here you can find some information, how to get the most out of Marvin JS User's Guide.

This user documentation consists of three pars:

  • Editor Overview
  • Drawing features
  • Feature overview pages

In Editor Overview section you can get to know the structure of the User Interface of Marvin JS. You can read detailed description about its elements, particularly the toolbars, context menus and dialogs.

This section is for you if you would like to get more information about Marvin JS; how this editor looks, or you just found a functionality and you are not sure what it is for, and you need a quick tip how to use it. 

 

Drawing features part includes all the information about the ways of drawing in Marvin JS. 

This section is for you, if you would like to know in which ways you can create, change or delete your structures, texts, graphical objects in Marvin JS.

 

In Feature overview pages we have collected all the functionalities which might be connected with a special workflow. (For example for searching Query feature set includes all the Marvin JS functionalities which might be useful for querying, etc.)

This section is for you, if you are interested in whether Marvin JS has functionalities related some special activities (e.g. handling Reactions, Query features, Stereochemistry, Markush structres).

Please, note that, as Marvin JS is a continuously evolving editor, our feature overview pages might change; and these pages only includes the special functions related to the specific features (but not all).
For the whole list of features, please, check the Drawing features or Editor Overview sections, or contact us: marvin-js-support@chemaxon.com .

Marvin JS : Copy

 

 Mutiply the molecule with Ctrl+Drag:

You can move a copy of the selected structure by pressing the Ctrl button then dragging the selected structure. When you do this, a "+" sign appears beside the cursor. A copy of the selected structure is placed on the canvas when you release the left button, and the original object remains selected, so it can be multiplied several times.

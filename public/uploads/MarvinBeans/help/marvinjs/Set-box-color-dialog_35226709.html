Marvin JS : Set box color dialog

In Set box color dialog you can change the color of the box graphical object. The outline and the background color can be set separately.

 

How to change the color of a graphical box

The color of the background and the line is changeable by opening the Set box color dialog via the pop-up menu. 

Choose the appropriate colors from the lists.

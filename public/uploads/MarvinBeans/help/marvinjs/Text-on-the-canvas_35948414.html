Marvin JS : Text on the canvas

 

 

 

 

With Text tool  you can write on the canvas. After selecting this tool from the Tools toolbar, the Text dialog can be opened by left-clicking on an empty place on the canvas. In the dialog, further text types are available: such as bold, italic, subscript, superscript. From the editor there is also possible to remove all the formats from a selected part of the text (remove formatting from text), or inserting special characters (symbols).

Please note, that every formatting options should be performed on a selected text. 

After creation, the textbox can be moved by dragging when one of the selection tools or the Text tool itself is active.

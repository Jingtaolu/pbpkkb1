Marvin JS : Abbreviated groups dialog


 


How to put an abbreviated group onto the canvas


Open the Abbreviated groups dialog via Abbreviated groups button on the Template Toolbar. Select a group then close the dialog and the selected group appears on the tip of the cursor. You can place it on the canvas with a left-click.

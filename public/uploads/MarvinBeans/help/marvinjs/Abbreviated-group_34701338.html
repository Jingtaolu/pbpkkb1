Marvin JS : Abbreviated group

 Abbreviated groups

Opens Abbreviated groups dialog window, where you can choose the appropriate abbreviation.

 

How to create abbreviated groups


Open the Abbreviated groups dialog via Abbreviated groups button on the Template Toolbar. Select a group then close the dialog and the selected group appears on the tip of the cursor. You can place it on the canvas with a left-click.

 

 

Tick the Expand checkbox if you want to see the structure in the expanded form.

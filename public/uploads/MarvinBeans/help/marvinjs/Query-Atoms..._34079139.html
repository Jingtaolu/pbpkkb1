What kind of query atoms are available in Marvin JS? 

Query atom symbol Included Atoms

How to draw a query atom:

Choose the appropriate query atom type from the combo-box.
The selected atom appears on the tip of the mouse pointer.
You can put it onto the canvas as many times as it is needed. 

 

To draw a Pseudo atom you should open the Pseudo atom dialog. For more information please, check the Pseudo atom dialog section.

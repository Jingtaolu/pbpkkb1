How to draw an electron flow arrow from an atom to an atom
Select the preferred type of electron flow arrow button from the Tools toolbar.
Drag the electron flow arrow from the electron(s) to the target atom or bond. A grey feedback of the arrow appears. 

 

 

Note that drawing electron flow arrows started from lone pairs is possible only, when lone pair visibility is turned on in View settings.

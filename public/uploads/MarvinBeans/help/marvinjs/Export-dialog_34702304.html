Marvin JS : Export dialog

In the Export dialog you can save your structure to the selected format or an image.

Keyboard shortcut: Ctrl+S

Supported export formats in Marvin JS

Natively Marvin JS supports the MRV and MDL Mol(V2000) formats. Other export formats also available via web service.

ChemAxon Marvin Document (MRV)
MDL Molfile (V2000)

Available image formats: PNG, JPEG (JPG)

Zoom mode: fit, autoshrink

Background: for PNG format transparent and white, but for JPEG format only white background is available

You can define the size of the image by setting the Width and Height values in pixels.

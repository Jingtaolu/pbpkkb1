Marvin JS : Zoom

 Zoom in

Magnifies the structures on the canvas.

 Zoom out

Reduces the size of the structures on the canvas.

 Zoom all

Sets the optimal magnification to be able to view everything on the canvas.

Keyboard Shortcut: Enter 

 Zoom to selection

Increases the zoom ratio placing the selected object in the center. This tool is only available when there is an active selection on the canvas.

Keyboard Shortcut: Ctrl + Enter 

Modifying the zoom ratio

The zoom ratio can be changed by zooming with the mouse wheel. With this function, you can also virtually "move" the canvas.

(This function is browser-dependent.)

How to draw a graphical object

 

 

To draw a shape, you can choose the Rectangle or Ellipse tool on the Tools toolbar, then dragging the mouse cursor until the shape reaches the desired size. Grey feedback shows how the real box will look like.  

 

 

 

  The size, shape and color of the object can be changed afterwards too.

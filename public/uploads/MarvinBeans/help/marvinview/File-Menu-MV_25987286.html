MarvinView : File Menu MV

Open Loads your saved molecule file into Marvin and discard any unsaved changes to the molecule you were previously working with.
Open Image Tries to convert an image file to a structure using OSRA.
Recent Files List of recently used file names.
Close Finishes working with the currently open molecule.
Save Selection Saves the molecule being in the selected cell. If there is no cell selected, you can specify molecules to save by giving a single cell index or a range of cell indices.
Save All Saves all molecules to a different location or with a different file name or format.
Save As Image Save an image of the molecule in the required location in required format. A saved image cannot be opened for later editing in Marvin.
Print Prints an image of the current molecule.
Exit Saves GUI settings, preferences and My Templates before exiting the application.

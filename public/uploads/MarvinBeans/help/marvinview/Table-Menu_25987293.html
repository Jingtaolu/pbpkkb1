MarvinView : Table Menu

Options... Opens the Table Options  dialog for settings the layout of the MarvinView table and the displayed property fields.
Show Fields Displays the fields of additional properties saved in a structure file.
Show Molecule Name Displays the molecules names saved in a structure file.
Show Generated Name Displays the generated IUPAC names of the molecules.
Show SMILES The the SMILES strings of the molecules.

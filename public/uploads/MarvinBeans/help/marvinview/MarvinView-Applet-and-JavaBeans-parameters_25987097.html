MarvinView : MarvinView Applet and JavaBeans parameters

Parameters of the MView and JMView applets can be set using the <param> HTML tag. Parameters of the MViewPane JavaBean can be set using the the setParams function.
For parameter constants please visit the ParameterConstants and ViewParameterConstants API.

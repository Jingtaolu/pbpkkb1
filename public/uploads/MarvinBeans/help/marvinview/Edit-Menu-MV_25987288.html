MarvinView : Edit Menu MV

Copy Copies the selection to the clipboard.
Copy As Copies the selection to the clipboard in the specified format.
Copy As Smiles Copies the selection to the clipboard in SMILES format.
Paste Inserts the contents of the clipboard at the location of the cursor, without replacing selection.
Find Find a cell by its serial number or performs full text search.
Find Again Restarts the previous Find Command.
Structure Open the selected molecule in MarvinSketch for further editing.
Source Opens the Edit Source Window. You can view and edit the source in any of the supported file formats.
Preferences The Preferences dialog window allows you to change many of the MarvinView display settings, including error highlighting and object visibility.

MarvinSketch : Drawing Simple Structures

You can draw any sophisticated structure with MarvinSketch.However, with all of the drawing tools available, drawing basic structures may seem overwhelming to the new user.

This chapter introduces basic drawing techniques to help you create your first structures.
From this chapter, you will learn:

  • How to Draw Atoms
  • How to Draw Bonds
  • How to Draw Chains
  • Spouting
  • Merging Structures
  • Templates
  • Drawing stereocenters
  • Coordination compounds
  • Insert/Edit new structure (fragment)
  • Atom Label Editor

MarvinSketch : Version Number

Version Number

From Marvin version 2014.07.07.0 the format YEAR.MONTH.DAY.PATCH_NUMBER is applied.

From Marvin version 5.7 an identifier appears in the file name of any downloadable artifacts, that indicates the internal build number of the file. This identifier begins with _b and a number. It is automatically generated and helps to identify the file in the build system of ChemAxon.

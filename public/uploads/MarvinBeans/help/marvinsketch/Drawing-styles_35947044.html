Drawing parameters can be collectively set by applying a style.

MarvinSketch provides a built-in default style, includes a small set of journal styles based on drawing parameters of some well-known scientific publications. Additionally,MarvinSketch stores user-defined styles.

Loading a style

  1. Open the  'Document Style Settings' dialogue selecting  File < Document Style < Settings.
  2. Choose amongst the user-defined and the pre-defined styles in the 'Journal Styles' pane. The settings are visualized in the panes.
  3. Click 'OK'. The chosen style will be applied for the whole document.

Restoring default settings

  1. Open the  'Document Style Settings' dialogue selecting  File < Document Style < Settings.
  2. Click on 'Restore to defaults' button.
  3. Click 'OK'.

Applying drawing parameters

  1. Open the  'Document Style Settings' dialogue selecting  File < Document Style < Settings.
  2. Set parameters in the 'Font Settings' and the 'Size Settings' pane.
  3. Click 'OK'. The chosen parameters will be applied for the whole document.

Saving a style

  1. Open the  'Document Style Settings' dialogue selecting  File < Document Style < Settings.
  2. Select 'Custom' in the 'Journal Styles' pane.
  3. Set the drawing properties.
  4. Click 'Save'.
  5. Type a name for the new style in the 'Save Style' dialogue.
  6. Click 'OK'.

Existing styles can not be overwritten in the 'Save Style' dialogue.

 

All your own saved files will be stored under the chemaxon/journal_styles/ directory of your home directory and will be added to the list of journal styles.

 

A new style file can be added to the chemaxon/marvin/styles directory under the Marvin installation directory. This new style file has to be listed in file chemaxon/marvin/styles/styleFileList.properties.

The new style file will be copied to the chemaxon/journal_styles/ library in your home directory and appear in the drop-down list of the "Journal style" pane.

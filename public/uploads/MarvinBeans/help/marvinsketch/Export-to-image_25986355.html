MarvinSketch : Export to image

The Export to Image option in the File menu allows you to save an image of the molecule in the sketcher.

Images exported  in .jpeg, png and svg  format can be re-opened in MarvinSketch if the source was exported in the image file.

In these format the  Save source option which is enabled by default can be disabled on the Export window by ticking it off tin the corresponding pane.

Marvin supports the following image formats:

              • JPG
              • PNG
              • SVG
              • EPS
              • TIFF
              • BMP
              • EMF
Saved images cannot be edited in Marvin

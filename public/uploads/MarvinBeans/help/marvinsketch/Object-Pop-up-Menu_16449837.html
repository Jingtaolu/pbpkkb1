MarvinSketch : Object Pop-up Menu

This menu appears when the context is a graphical object like Text, Bracket, Electron Flow arrow, or other Graphics.

Menu Item

Description

Bring to Front

Brings the selected object in front of all others.

Send to Back

Places the selected object behind all others.
Delete Deletes the object.
Transformation Main menu for the Flip / Mirror / Invert submenu.
Format... Brings up the Graphics Object Properties dialog where you can select from several formatting options.

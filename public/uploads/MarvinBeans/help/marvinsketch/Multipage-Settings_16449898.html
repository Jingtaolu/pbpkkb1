MarvinSketch : Multipage Settings

In MarvinSketch a document is the canvas where you can create and manipulate structures.

A document may contain multiple pages, or a single page according to the settings in Document type panel.

Additionally you can set

  • the number of horizontal and vertical pages in the Document Grid part
  •   a title (optional,"molecule title" property) in the Document Header textbox.
  • the page size
  • the margins in the corresponding sections of this dialog window.

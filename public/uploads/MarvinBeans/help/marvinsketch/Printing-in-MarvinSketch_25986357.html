MarvinSketch : Printing in MarvinSketch

You can print an image of the current molecule by choosing File > Print.

If you print from a single page document and the size of your molecule(s) is bigger then the paper size, it will be shrunk to one page.

If you want to print your molecule(s) to multiple pages change the document type to multipage document in File > Multipage Settings..., and arrange the structures on the pages.

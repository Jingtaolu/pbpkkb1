MarvinSketch : Service Implementations

Although the most common services are already implemented, Marvin Services provides an extensible API to implement custom services.

Please note that some implementations has type restrictions handled by the built-in editor.

The supported service protocols listed below:

 
  • Local Service
  • WSDL/SOAP
  • XML-RPC
  • JSON-RPC
  • HTTP

MarvinSketch : How to add, edit and format text

 

In addition to atom labels, you can also add blocks of text in textbox.
Textboxes
  • are resizable
  • they are outlined by a border
  • they have a background.
  •  are not fitted to the text automatically.

Creating a textbox

  1. Click on the Text tool  in the Tools toolbar or use Text item  in the Insert menu.
  2. Click and hold the left button, move the mouse and release the button.
  3. To place the textbox click again.
  4. Type a text.

The shape textbox is changeable or resizable by dragging one of its points or the selection.

Editing and formatting  text

When creating/editing the text area, text format functions will be provided in a toolbar that appears under the template bar.

 

Text format functions:

  • Font type
  • Font size
  • Bold
  • Italic
  • Color
  • Subscript
  • Superscript
  • Symbols

To edit a text select the Text tool and click on the text to be modified.

 

The character map is active only when a text box is open for editing.

With a text box open, mouse over the character map to select one to be inserted. The 16 most recently used characters appear at the bottom of the More Symbols window.


You can place a text box with the IUPAC name(s) from the Structure > Structure to Name > Place IUPAC Name menu command and it will be automatically inserted under the structure. The name will be updated in real-time.

 

Text can be formatted via the Text Attributes pane of the Graphic Object Properties window.

Structure Checker : Structure Checker in a Nutshell

Structure Checker is a tool for detecting drawing mistakes or special structural elements in chemical structures.  Besides this task, the unpreferred representations can also be fixed by Structure Checker if needed.

Structure Checker works with predefined checking options and you can choose which ones should be considered when you run a check. Additionally, you can also define a fix to change the reported issue in the structure.

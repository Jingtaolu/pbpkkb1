Structure Checker : JChem Cartridge

Setting external checkers in JChem Cartridge

In order to enable external checker usage in JChem Cartridge, the following line should be included to the  jchem/cartridge/conf/jcart.properties file:

sysprop.chemaxon.structurechecker.factory.ExternalCheckerFixerConfiguration=<path-to-externalcheckersfixers.xml>

The provided xml file contains the configuration of the external checkers and fixers to be used.

To create this configuration xml, use Marvin Sketch or the Structure Checker Application as described.

Using external checkers in JChem Cartridge

The external checkers and fixers can be used with the jc_evaluate_x Cartridge operator by the check and fix Chemical Terms functions.

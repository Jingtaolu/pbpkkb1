Licensing : License Report

We've introduced license report generation in 6.3.0.

A license report contains all the relevant information about the configuration of ChemAxon's licensing packed into a singe zip file.

The collected data:

  • Version of the installed software
  • System and Java properties regarding ChemAxon license configuration and the files they are configured to
  • .cxl files from the chemaxon folder in the user home
  • .log files from the chemaxon folder in the user home

To generate such report, start the License Manager and select the "Create report" option and click the "Create report" button.

Then select the folder where the report should be generated into and send the licenseData.zip to us via e-mail.

License report can be generated via the license console tool using the -r option.

File Formats : Default valence of metal atoms

Metal atoms are exported with their valence electrons into MDL mol format.

Consequently, 1 or 2 implicit hydrogens are added to the alkali metals and alkaline earth metals upon export.

By default Tin(Sn) is exported with 2 valence electrons and Thallium (Tl) with one.

You can set a non-default valence using the Atom menu or the Atom pop-up menu.

File Formats : BMP

ChemAxon products are able to export BMP (Microsoft Bitmap) image files. The MsbmpExport module recognizes the common image export options.

Examples:

msbmp Default settings: 200x200 pixels, white background (or black in 3D).
msbmp:w100,#ffff00 100x100 BMP with yellow background.

Codename: msbmp

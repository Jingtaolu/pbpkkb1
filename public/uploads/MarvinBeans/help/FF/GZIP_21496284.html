File Formats : GZIP

Codename: gzip

ChemAxon's products are able to import and export GZIP compressed molecule files. At export, the molecule file format is specified as an export option.

Examples:

gzip:sdf Exports a gzipped SDfile.
gzip:smiles:0 Exports gzipped SMILES without stereo information.
